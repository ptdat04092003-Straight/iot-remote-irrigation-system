# IoT Remote Irrigation System
## Overview
This project is an IoT-based remote irrigation system that allows monitoring and controlling a water pump remotely.
## System Architecture
Sensor → PLC → Node-RED → C# Application → User
## Components
### PLC
- Controls pump operation
- Processes sensor signals
### Node-RED
- Handles IoT communication
- Transfers data between PLC and user interface
### C# Application
- Provides monitoring interface
- Allows remote control
## Features
- Remote irrigation control
- Automatic operation based on sensor data
- Real-time monitoring
